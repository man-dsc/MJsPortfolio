Web Scraping

Web scraping is used to extract data from websites, enabling the collection of large datasets for analysis. Data mining involves processing and analyzing this scraped data to uncover patterns and insights. Using input data, such as car specifications or location, we can identify similar cars available online by comparing features and attributes.

Building a Baseline

To address the chance of insufficient data for machine learning training, an average price is calculated from the scraped data. The scraped data may not represent the exact same car but allows for variability in factors like odometer readings and model year. By accounting for these differences, a reasonable average price can be determined to provide a baseline. This approach ensures that some value is formed, enabling the process to proceed even with limited direct data.

ML Training and Testing

The machine learning model is trained using linear regression, leveraging price and mileage as key features. This approach identifies the relationship between a car's mileage and its price, allowing for predictive analysis. As more data becomes available, the model's accuracy improves, refining the estimation of price per kilometer. This ensures more reliable insights, enabling better decision-making based on mileage and pricing trends.
